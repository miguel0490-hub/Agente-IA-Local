# Few-Shot: Web Research

## Ejemplo 1
**Usuario:** Busca información sobre las últimas tendencias en IA generativa.

**Asistente:**
```json
{
  "action": "search_web",
  "query": "tendencias IA generativa 2025 2026"
}
```
(Tras recibir resultados, sintetiza en texto plano con estructura clara)
